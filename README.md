# 🗄️ ApexPlanet Data Analytics Internship – Task 2

## 📌 Project Title

**SQL & Data Extraction using Python and SQLite**

---

## 📖 Overview

This project is part of the **ApexPlanet Software Pvt. Ltd. Data Analytics Internship**.

The objective of this task is to understand SQL fundamentals, perform data extraction and aggregation, and integrate SQL with Python using SQLite. The project demonstrates how to load a dataset into a database, execute SQL queries, and analyze business data efficiently.

---

## 🎯 Objectives

* Import a CSV dataset into an SQLite database.
* Execute SQL queries for data analysis.
* Perform data aggregation and filtering.
* Connect Python with SQLite.
* Extract SQL query results into Pandas DataFrames.

---

## 📂 Dataset

**Dataset Name:** SuperMarket Analysis Dataset

The dataset contains supermarket transaction records, including:

* Invoice ID
* Branch
* City
* Customer Type
* Gender
* Product Line
* Quantity
* Sales
* Payment Method
* Rating
* Date & Time
